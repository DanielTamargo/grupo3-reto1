ESTADISTICAS

Disponemos de una serie de gráficas que se actualizan en tiempo real para consultar los datos constantemente. De esta manera, podremos observar todos los datos registrados* y analizarlos en pos de mejorar el servicio que ofrece EuskoTren.


*registrados:
Realmente no están registrados, debido a que no disponemos de una BBDD donde guardar registros, son datos que se generan aleatoriamente.
De hecho, se generan a una velocidad no natural con la intención de que se aprecie el resultado de las gráficas al cambiar dinámicamente con el flujo de datos en tiempo real.
